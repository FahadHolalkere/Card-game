This Project has two files:
emitrr folder is frontend file which is react-app
Backend folder is Go lang backend part

you should have go and nodejs installed in your system before all these steps
steps:-
1.open emitrr file in vscode and open another window in vscode for opening the Backend folder
2.Now,in emitrr folder terminal run command "npm run start"
3.In Backend folder terminal run command "go run main.go"
4.When you run go command it may ask you to allow main.exe to run and allow firewall to access, don't worry and allow access.(if didn't ask skip)
5.In Browser, type localhost:3000 which will render the registration page.
6.Enter name, hit submit and press Start game and that's it enjoy the game
